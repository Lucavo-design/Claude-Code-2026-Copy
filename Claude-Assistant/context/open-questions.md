## Answered Questions

### Q: Where does Claude Code store its configuration?
**A:** In `~/.claude/` folder on your Mac. This includes settings, allowed tools, and MCP server configs.

### Q: Can different Claude products share the same files?
**A:** Not automatically, but you can set up a shared folder and point each product to it using different methods (--add-dir for Claude Code, Filesystem MCP for Claude Desktop, etc.)
