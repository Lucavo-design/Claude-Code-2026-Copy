## How It Works

When starting a Claude Code session in this directory, Claude automatically reads the context files to understand:
- Your communication preferences
- Current projects and priorities
- Tools and accounts you use
- Your skill level and learning goals

## Getting Started

1. Clone this repository
2. Open Claude Code in the project directory
3. Claude will automatically load your context from `Claude-Assistant/CLAUDE.md`
4. Start working on your projects with personalized AI assistance
